# ⚡ Amilma Network - Speed Test
> Professional Network Intelligence

Moderna, brza i precizna web aplikacija za mjerenje brzine internet konekcije. Dizajnirana sa fokusom na savršeno korisničko iskustvo, tačnost podataka i fluidan UI.

---

## ✨ Ključne funkcionalnosti

* **Precizno mjerenje:** Analiza *Download* i *Upload* brzine u realnom vremenu uz pomoć naprednih JavaScript algoritama.
* **Napredna metrika:** Praćenje *Pinga* i *Jittera* za potpunu sliku o stabilnosti i kvalitetu mreže.
* **Moderan UI/UX:** Intuitivan i čist interfejs sa animiranim brzinomjerom koji reaguje na promjene protoka.
* **Optimizacija:** Lagana arhitektura bez eksternih "teških" biblioteka, zasnovana na HTML-u, CSS-u i Vanilla JavaScript-u.
